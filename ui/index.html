Northwestern PC Support File Distribution GUI

This app copies files/folders from your local system and distributes it to computers in the selected labs.
Note that you need domain-level access to use this tool.
To find out more, please consult the README that should have come with this executable.

Leave the system selection prompt blank to copy onto every system in the selected lab.
